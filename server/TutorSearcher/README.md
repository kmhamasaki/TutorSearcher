# Getting Started

## set up your IDE

1. clone the repo
2. open eclipse
3. file > import > maven > existing maven project and select `server/TutorSearcher` and import it

## install maven 

https://maven.apache.org/download.cgi

# to run

in the `server/TutorSearcher` directory run

``mvn clean install``

``mvn spring-boot:run``
